Automated Micro-Aneurysm Detection App for Diabetic Retinopathy

Diabetic retinopathy is a serious complication of diabetes that can lead to vision loss if not detected and treated early. Manual examination of eye fundus images for signs of diabetic retinopathy, such as micro-aneurysms, is time-consuming and subjective. To address this, we have developed an automated solution using deep learning techniques.

App Functionality

The app utilizes a deep learning model called MAMC-Net (Multi-Resolution Attention and Multi-Scale Convolution Network) to automatically detect micro-aneurysms in eye fundus images. This model is based on the U-Net architecture and incorporates specialized components to improve accuracy.

Model Architecture

  • The MAMC-Net follows an encoder-decoder structure with skip connections.
  • Encoder: Extracts low-level features from input images using convolutional blocks, max-pooling layers, and dropout layers. It generates multi-resolution feature maps to capture global contextual information.
  • Decoder: Utilizes up-convolution layers, multi-resolution attention modules, and multi-scale convolution modules. These components help in accurately segmenting micro-aneurysms by integrating spatial and channel-wise attention and capturing features at multiple scales.

Multi-Resolution Attention

The Position Attention Module captures spatial inter-dependencies between features. The Channel Attention Module aggregates features across channels to focus on specific semantic features. These modules combine to produce a multi-resolution attention map used for micro-aneurysm detection.

Multi-Scale Convolution

Designed to capture features at multiple scales by combining information from different pooling sizes. Features are processed by separate convolutional layers and upsampled to create a comprehensive representation. This facilitates the accurate segmentation of micro-aneurysms by capturing both global and local information.

How to Use

  1. Copy the provided model architecture into a file named unet.py.
  2. Modify the build_model function in unet.py to return your desired model.
  3. Save the model state dict as Model.pt and place it in the Model folder.

Conclusion

The automated micro-aneurysm detection app offers a promising solution for early diabetic retinopathy detection, potentially preventing vision impairment and blindness.
